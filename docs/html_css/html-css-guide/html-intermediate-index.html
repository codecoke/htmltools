HTML中级指南

您的位置：首页>>教程>>HTML 标准指南>> > HTML中级指南

鉴于HTML初级指南的目的只是教授HTML的骨架元素，这个中级教程增加一些骨肉，它们不应该特别的难，但可以进一步增进我们对HTML的理解，使我们能多做一些更有用的事情。

HTML中级和高级指南的各部分并没有特别的顺序，每页都是独立的，你可以按你的需要、意愿学习不同的部分。

目录

  • span和div──“无意义”的元素，用于CSS。
  • 元标签──运用页面的额外信息。
  • 表格──HTML初级指南的表格教程的扩展，关于表头和单元格的合并。
  • 定义列表──被忽略的列表。
  • Javascript──如何在你的HTML中使用Javascript。
  • 不良标签──你应该避免的HTML元素和属性。

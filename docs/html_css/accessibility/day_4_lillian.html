第四天：马大姐

lillian 第四天：马大姐

据说 马大姐 刚从香港搭船移民来贵阳。其实她是 30 年前搭飞机来的，不过人们还是如此传颂着。她今年 54 岁，跟女儿、养子和两岁大的孙子寡居于猫眼胡同，她们大概就花去了她所有的时间了。虽然她的女儿和养子都熟稔于两种方言，但是 马大姐 的普通话还是支支吾吾，不如广东话来得流畅。不跟孙子一起玩的时候，她也会试着读报纸来练普通话；她会把报纸摊开在餐桌上，打开天花板上的 100 瓦大灯，用放大镜慢慢地读。

马大姐 在一个居委会里担任行政助理。为了省事，这个居委会和没办法给住在石景山的二十多岁色盲莫非提供高速网络的是同一个。不过 马大姐 并不知道这件事。她们的技术部才刚大规模地转移到 Windows XP 和 Internet Explorer 6 的环境里，而且接下来的九个月还会更为乖戾。他们同时采用了新的网络安全政策：不准使用 Java 、不准使用 Javascript 、不准使用 Flash 也不准使用 ActiveX 控制元件，除非经过了他们自己批准。当时他们显然还没有批准任何网站。技术上来说，这表示所有的网站在 Internet Explorer 的安全性页面里，都是“限制的网站”，所以所有的脚本都被关闭了；你得拥有系统管理权限才有办法把某个网站加入“信任的网站”里。

不消说， 马大姐 自然是没有系统管理权限了。

不过她能用 19 寸的大荧幕来跟技术部的 固赢 的苦口婆心劝阻相抗衡。他常常想起那一次因为电梯坏掉，所以他得帮 马大姐 把大荧幕努力扛到楼上去；所幸他很少不耐烦地抱怨这件事。 马大姐 其实还挺喜欢 固赢 的，他可以说是那群人当中最善良的一个了，甚至还帮她把 Internet Explorer 里的字型调到“最大”，好让她女儿博客(blog)的字大到能被她阅读；因此她每天都会去看。但是当她问 Matt 为什么 CNN.com 的字就没办法调得更大时， 固赢发表了一次关于浏览器大战的演讲，扔出一大堆专业词汇，最后仍旧只能说他实在无能为力。

马大姐 好希望可以阅读更多网站；但既然 固赢 没办法修的话，就没有人有办法。

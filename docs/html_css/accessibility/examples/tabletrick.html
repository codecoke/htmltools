My Weblog, with table trick

Today's posts

This text will be displayed first in Lynx and read first in JAWS, even though it is visually positioned to the right of the navigation bar. This benefits blind users, and it also benefits you, since search engines give higher precedence to keywords that come earlier in your page.

  1. The non-tricky version of this page. Same layout.
  2. The non-tricky version in Lynx. Yuck.
  3. The table trick in Lynx. Much better.

Here's the basic concept. Instead of the obvious table layout:

<table>
<tr>
  <td valign="top" align="left" width="25%">
    ... navigation bar ...
  </td>
  <td valign="top" align="left">
    ... main content ...
  </td>
</tr>
</table>

Do this instead:

<table>
<tr>
  <!-- spacer GIF in upper-left cell -->
  <td><img src="../images/1.gif" width="1" height="1" alt=""></td>
  <!-- main content cell first, with rowspan=2 -->
  <td valign="top" align="left" rowspan="2">
    ... main content ...
  </td>
</tr>
<tr>
  <td valign="top" align="left" width="25%">
    ... navigation bar ...
  </td>
</tr>
</table>

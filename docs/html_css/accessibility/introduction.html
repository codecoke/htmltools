introducing 前言

这本书定名为《深入亲和力：在 30 天内打造更具亲和力的网站》，它将会回答两个问题：第一个问题是“为什么该让我的网站更具亲和力？”如果你根本就没有自己的网站的话，显然就不会成为本书的读者了。第二个问题是“如何让我的网站更具亲和力？”如果第一个问题的回答说服不了你，那么第二个问题大概也将无法引起你的兴趣。

为了回答第一个问题，我会先介绍以下五个人的特征素描：小莉、莫非、老路、马大姐及陈建国。这些人有几个共同点：

  1. 她们都有身体上、心理上、以及技术上的障碍，使得网际网络的使用更为困难。
  2. 虽然她们都是虚构人物，却能反映出现实生活中被障碍所困扰的人们；同时她们也以相对应的方式使用网际网络。
  3. 她们浏览你的网站时，都会遇到困难。

为了回答第二个问题，我会介绍 25 个能够立即套用在你网站上的技巧，让它更具亲和力。虽然这些概念可以应用到所有网站上，但我只会专注在几种热门博客(blog)工具 (weblogging tools) 的实作方面。如果你使用的是其他的出版工具或模版系统的话，就得确定如何在那些工具上实作这些技巧。

每个技巧都会着重在单一概念上；我会解释它背后的原因，并指出谁会因为你实作而获益。这也是我们先谈到特征素描的原因，因为这能把第一个问题的语调从“为什么我得烦恼这些？”转变为“谁将因此而获益？”于是我们可以回答“陈建国 将能从中获益。”“为什么 陈建国 能从中获益？”“好，让我们从这点来看......”，如此进行。

即使你不是 HTML 专家也甭紧张。就算你的网站只是个人博客(blog)，而且打从你选好模版的第一天后就没再去碰过它们，也别太担心。我不是要你彻底地重新设计网站，一边咒骂你那些巢状表格，一边把它们转换成 XHTML 与 CSS 语法。我们只不过是要聊聊你现有的基础，然后利用一些小巧却重要的步骤让它们变得更好。小莉、莫非、老路、马大姐 及 陈建国 都会为你的体贴入微而感动不已。

第二天：莫非

michael 莫非

莫非现年 30 岁，与女朋友 胖丁 住在石景山的 古城 。 他 28 岁的时候学习设计网页并创办ibar。两年后，他在洽当的时机脱手，卖给一家全球性的腐败集团；同时这家集团也用电话解释给 莫非 听，让他知道为什么不能让他那间一个月 1300元的公寓有比 56K 拨接更好的服务。看来金钱也有不是万能的时候。

莫非 眼下的世界跟你所看到的不同。这可不是甚么商业的陈腔滥调或是哲学声明；他看到的世界真的跟你所看到的不一样。他是红绿色盲，没有办法分辨红色跟绿色（某些红色看起来会非常深，近似于黑色）。他所有的衣服都标上字母来区分颜色，像是用 R 代表红色、 DG 代表深绿等等。他的女朋友设计了一套搭配组合，让他知道哪些衣服可以搭配在一起。他按照这些字母的指示来穿着，也如此表达；但他不能分辨其间的差异。

当客服人员温言告诉他说他再也不需要工作的那当儿， 莫非 正一如往常地在网络上。现在他退休了，开始把时间都花在搜集老旧的大型游戏机台上，然后试着把这些游戏机台修好，放进橱柜里、加以详列说明，最后再到拍卖网站上把他们卖掉。事实上大部分的细节都是他女朋友做的，不过其他所有的事都还是 莫非 在处理。他还养了一只叫 可乐 的猫；因为 胖丁 抱怨可乐太调皮了，所以他们妥协后，决定按照她的意见。 可乐 大部分的时间都在沙发的角落睡觉；不过早上交通颠峰期的时候，它就会坐在窗枱上看着窗外的车水马龙。事实上 莫非也会。 胖丁 大概是唯一选择继续睡觉的人了。这是专属于他和可乐猫的独处时间。

正如你所料， 莫非 的嗜好仰赖大量的线上互动。他得跟那些客户们（不论是过去或是未来所遇到的）、供应商们、机台拥有，人以及任何可能在机台零件上有着卓越见解的人打交道。他实在恨透了他的 56K （实际上只有 28.8K ）拨接了，这让他的浏览器几近爆炸。某些日子里，他会用一个叫 Links 的纯文字浏览器，这个浏览器就跟其他的图形浏览器一样，可以画出复杂的多栏版面，不过就只有文字，而没有图片或字形变化。另一些日子里，他也会用 Opera 浏览器，因为这个浏览器允许他在背景载入图片，而且可以很方便地开启或关闭图片选项；他多半会因为频宽的考量把他们关闭。今天则是使用 Links 的日子。

红绿灯由上而下分别是红、黄、绿；可是横向的红绿灯位置却不一定，因此相当危险。 莫非 就跟其他8%有色盲的人一样，其实偷看了附近其他人的动作，才知道离地最近的是绿灯、最远的是红灯，而中间的是黄灯。

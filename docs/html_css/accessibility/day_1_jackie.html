第一天：小莉

Jackie第一天：小莉

王莉莉 （昵称 小莉 ）是个成绩优异的 19 岁高中生，和妈妈住在深圳 。她的成绩一部分来自于私人家教的协助，但主要还是因为她勤奋学习及广泛阅读。她所有的课本都是有声书，使用特殊的录音机来拨放；这种录音机的拨放速度比正常快 3 倍，却只会造成一点点失真。她失明已经 8 年了。

小莉 并非天生失明，所以她还能理解像色彩这样的视觉概念，而且仍然会跟妈妈聊到“以前”生活里的色彩。她唯一不谈的是害死她爸爸、让她失明的那场车祸；这在她使用“以前”跟“之后”两个词汇时可以看出些许端倪。“就像以前客厅墙壁的那种绿色。”“这像你以前的那件紫色毛衣，不过更亮一点。”诸如此类。

由于 小莉 努力地调适生活，以致于她的学业进度比同学们慢了两年。今年秋天她将进入北京大学，主修比较文学。她对此感到相当兴奋，一方面是希望交到新朋友，另一方面则是因为能在线上完成更多课堂作业：阅读课程规划，提出论文，与教授、同学们即时传讯等等。她高中的朋友并不多，多半是跟妈妈还有网际网络在一起。

她每个月花在有声书、音乐以及一堆小玩意上的钱都超过 300 美金。虽然有越来越多吸引人的读物能够下载到电子书阅读器里，但是她大部分的有声书仍旧是录音带。她那些音乐 CD 以及小玩意儿都是从线上零售商买来的。

线上购物未必简单，但是跟自己搭车到卖场然后等店员招呼比起来，实在是好太多了。线上购物也是少数几件她不需依靠导盲犬，就能完成的事情。她不喜欢现在这只导盲犬 Arthur ；它不像之前几只那么优秀。 Lancelot 跟 Guinevere 都已经退休了，现在跟她们住在一起。她靠项圈来分辨她们： Arthur 的项圈是光滑的、 Lancelot 的带刺， Guinevere 的则有凹槽。

就像大部分的盲人一样，小莉 对布拉耶 (Braille) 点字法了解不多。她有一台点字标签机可以帮 CD 贴标签，但她没有办法阅读点字书，因为所有的点字书都是用第二级的点字法写的，而她从来没学过。她用最新版的 JAWS 来线上购物及活动，这是一套能在视窗环境下与 Internet Explorer 整合的屏幕阅读软件。 JAWS 使用了先进的电子语音合成器来朗读网站内容，也提供一堆深奥的键盘快捷键来浏览网站。小莉 记得所有按键，所以只要网站设计良好，她阅读网站的速度就能比阅读有声书还要更快。

第五天：陈建国

Marcus 第五天：陈建国

陈建国 的父母吸毒，所以他天生视盲。他早产了三个月，在保温箱里住了六周，只有 10% 的机会能够活下来；不过他现在已经 31 岁了。他在一家 移动客服中心工作，帮忙在聋人及听人之间转译电话。他们是这样运作的：
  1. 假设有个聋人或听障的人，我们就先叫她 美莎 好了，用她的 ADSL 或电脑拨号到中心来。
  2. 美莎 键入她想要交谈的对象的名字和号码；就先假设那个人叫 中移动 吧。
  3. 陈建国 就会打电话给 中移动 ，然后告诉他有个叫 美莎 的人从转译中心要打电话给他。
  4. 不论 美莎 键入了甚么， 陈建国 都会从他的 ALVA 可覆新布拉耶点字显示器 上读到。任何 美莎 输入的字都会被即时地转换成布拉耶点字。
  5. 不论 中移动 说了甚么， 陈建国 都会以每分钟 110 字的神速打给 美莎 看。

无论是 美莎 或 中移动 都不知道（其实也不在乎） 陈建国 是盲人。唯一的线索就只有 ALVA 的针上下移动时所发出的规律噪音，听起来有点像远处打字机所传来的声音。

陈建国 的眼睛跟大脑之间真的甚么都没有了。虽然如此，每个下午当太阳光从窗户射进他的隔间时，他还是会抱怨着因此头痛不已，然后把百叶窗拉上。

陈建国 在家里的时候也会拿 ALVA 搭配全荧幕 DOS 视窗里的 Lynx 纯文字浏览器使用。他在家里读这些网页的方法，就跟他工作时的作法一样：用布拉耶点字，一次读一列。他讨厌屏幕阅读软件，就算有一套也不想听；因为他总是习惯一回到家就把收音机转到最大声，直到睡前才关上。

当他听着收音机在网络上漫游时，他也会用电话跟其他人聊天。他每周至少都会跟 100 个人讲过电话；这听起来很不寻常，不过他自己却完全不晓得。如果你曾经玩过“六度分离”这个游戏，试图寻找你跟世界上任何另一个人的关联时，你很可能会发现 陈建国 就是你的六度亲人之一。他就是那种三姑六婆型的人，即使是最有经验的市场老手也未必能够打断他的话；就是因为有他这种人，所以你的电话公司才会不打算提供无限市内通话的选择。他热衷于告诉亲朋好友们，刚刚看到了多酷的网站。事实上他刻意地使用了“看到”这个字眼，因为他晓得这会让他们觉得不自在。

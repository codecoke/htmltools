display

  • 属性清单
  • display
版本：CSS1/CSS2　 兼容性：IE4+ NS4+　继承性：无

语法：

  • display : block | none | inline | compact | marker | inline-table | list-item | run-in | table | table-caption | table-cell | table-column | table-column-group | table-footer-group | table-header-group | table-row | table-row-group

参数：

  1. block : 　CSS1　块对象的默认值。用该值为对象之后添加新行
  2. none : 　CSS1　隐藏对象。与visibility属性的hidden值不同，其不为被隐藏的对象保留其物理空间
  3. inline : 　CSS1　内联对象的默认值。用该值将从对象中删除行
  4. compact : 　CSS2　分配对象为块对象或基于内容之上的内联对象
  5. marker : 　CSS2　指定内容在容器对象之前或之后。要使用此参数，对象必须和:after及:before 伪元素一起使用
  6. inline-table : 　CSS2　将表格显示为无前后换行的内联对象或内联容器
  7. list-item : 　CSS1　将块对象指定为列表项目。并可以添加可选项目标志
  8. run-in : 　CSS2　分配对象为块对象或基于内容之上的内联对象
  9. table : 　CSS2　将对象作为块元素级的表格显示
  10. table-caption : 　CSS2　将对象作为表格标题显示
  11. table-cell : 　CSS2　将对象作为表格单元格显示
  12. table-column : 　CSS2　将对象作为表格列显示
  13. table-column-group : 　CSS2　将对象作为表格列组显示
  14. table-header-group : 　CSS2　将对象作为表格标题组显示
  15. table-footer-group : 　CSS2　将对象作为表格脚注组显示
  16. table-row : 　CSS2　将对象作为表格行显示
  17. table-row-group : 　CSS2　将对象作为表格行组显示

说明：

  • 设置或检索对象是否及如何显示。
  • 目前 IE5.5仅支持以上CSS1的参数。
  • 对应的脚本特性为display。

示例：

img { disply: block; float: right; }

效果：

旺财您所看到的文字块的名字是小强，图片的名字是旺财，请您从下方的选择框内选择旺财的display属性的不同的值，看一看会发生什么，然后您就会明白这个属性的意义。希望您喜欢CodeCoke。谢谢。

marker-offset

  • 属性清单
  • marker-offset
版本：CSS2　 兼容性：NONE　继承性：无

语法：

  • marker-offset : auto | length

参数：

  1. auto : 　浏览器自动设置间距
  2. length : 　由浮点数字和单位标识符组成的长度值。可为负值。请参阅长度单位

说明：

  • 设置或检索标记容器和主容器之间水平补白。即两个容器靠近的一边的间距。
    目前IE5.5尚不支持此属性。
    对应的脚本特性为markerOffset。

示例：

li:before { display: marker; marker-offset: 5px; }

结语

网页亲和力 结语

以上就是“深入亲和力：在 30 天内打造更具亲和力的网站”。谨代表 小莉 、 莫非 、 老路 、 马大姐 和 陈建国 感谢您对此议题的关注。

延伸阅读：我推荐的阅读书目

  1. Joe Clark: Building Accessible Websites. 我是这本书的技术编辑，这真的是本博而不滥的好书。
  2. Jim Thatcher and others: Constructing Accessible Web Sites. 比 Joe 的书精简一点，但是它所囊括的主题深度更为过之。这本书还提供了许多的撷取画面，展示出各种屏幕阅读软件及浏览器在各种不同的标签及标记上所显示的效果；同时还有一个章节描述着跟亲和力需求有关的法律声明。
  3. Steve Krug: Don't Make Me Think: A Common Sense Approach to Web Usability. 可用性及亲和力在许多方面都会相重叠； Steve 在可用（且亲和）的导航上提出了许多很棒的忠告。
  4. Owen Briggs, Steve Champeon, Eric Costello, Matthew Patterson: Cascading Style Sheets: Separating Content from Presentation. 就如同我们在这一系列的课程理所看到的一样， CSS 乃是亲和力中不可分割的一部份，因为他能让你在 HTML 标记（这也是辅助科技所在乎的部分）里“做正确的事”，而仍旧让你在视觉性浏览器里做出你想要的效果。
  5. Eric Meyer: Eric Meyer on CSS: Mastering the Language of Web Design. 在这本书的指南网站上也有一些免费的材料。

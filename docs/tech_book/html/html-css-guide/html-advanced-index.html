HTML高级指南

您的位置：首页>>教程>>HTML 标准指南>> > HTML高级指南

HTML高级指南的目的是开拓HTML的优点，优化亲和力和易用性。

因为HTML初级指南和HTML中级指南持使用最新HTML标准（XHTML 1.0以上版本）的立场，可能某些没有读过初级和中级指南的人对这个指南的某些方面感到难以理解，这也是标准要点存在的原因。总体上，这个指南的各部分都假设你对HTML已经里外理解，并要进一步开发它的潜能。

目录

  • 标准要点──分离内容与表现，XHTML的标签、属性和易用性的概括。
  • 亲和连接──使超连接对残障人士更具亲和力。
  • 精通文本──为正确的内容使用恰当的标签。
  • 精通表格──列组合，表头和表注等等等等。
  • 亲和表单──使表单对残障人士更具亲和力。
  • 声明──声明文档类型，语言和字符集。

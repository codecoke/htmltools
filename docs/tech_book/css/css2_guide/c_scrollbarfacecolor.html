scrollbar-face-color

  • 属性清单
  • scrollbar-face-color
版本：IE5.5+专有属性　继承性：有

语法：

  • scrollbar-face-color : color

参数：

  1. color : 　指定颜色。请参阅颜色单位和附录：颜色表

说明：

  • 设置或检索滚动条3D表面（ThreedFace）的颜色。
    请参阅overflow属性 。
    对应的脚本特性为scrollbarFaceColor。

示例：

div {scrollbar-face-color : threedface; }
